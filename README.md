# Selenium WebDriver Multiple Frames Handling

This project demonstrates how to work with multiple iframes using Selenium WebDriver, Java, and TestNG. It covers switching between frames, navigating across web pages, automating login forms inside frames, and browser navigation techniques.

## Features

- Count the total number of iframes on a webpage
- Switch between multiple frames using frame names
- Automate login forms inside iframes
- Navigate between local HTML pages and external websites
- Perform browser navigation using `navigate().to()`
- Cross-browser support using a reusable Helper class
- Browser automation with ChromeDriver, EdgeDriver, and FirefoxDriver

## Test Scenarios

- Count available iframes in a webpage
- Switch to specific frames by name
- Perform login automation inside iframes
- Navigate between multiple websites
- Capture and print page titles after navigation

## Tech Stack

- Java
- Selenium WebDriver
- TestNG
- ChromeDriver
- EdgeDriver
- FirefoxDriver
- Maven
- Eclipse IDE
